# WOD Timer (PWA)

A small offline-capable web app you can add to your iPhone Home Screen.
- Browse workouts
- Open the relevant PDF page (as an image for reliability on iOS)
- Run a count-up or countdown timer

## Deploy via GitHub Pages
1) Create a GitHub repo (public is easiest).
2) Upload all files from this folder.
3) Repo Settings → Pages → Deploy from branch → main / root.
4) Open the GitHub Pages URL in Safari on iPhone → Share → Add to Home Screen.
